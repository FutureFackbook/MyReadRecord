## Update History

**2016.12**

- 《囚徒健身》保罗威德 关于用自身重量健身的书。有很强烈的冲击与指导意义。
- 《囚徒健身2》保罗威德 一些细小肌肉的锻炼以及关节，还有一些心理方面的指导。这部分适合《囚徒健身》中六艺达到一定水平之后练习。

- 《断舍离》山下英子
